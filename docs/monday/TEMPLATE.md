# PR Merge Documentation Template

This template is used by the Monday-GitHub sync workflow to generate
documentation updates posted to Monday.com items when a PR is merged.

---

## Sections

### Summary
Brief description of what was implemented/changed in this PR.

### Files Modified
List of files added, modified, or deleted.

### Key Design Decisions
Any architectural or design choices made during implementation.

### Testing
- Tests added/modified
- Test results (pass/fail count)
- Manual testing notes

### PR Reference
- PR Number: #{pr_number}
- PR URL: {pr_url}
- Branch: {branch_name}
- Merged by: {merged_by}
- Merge date: {merge_date}
